## 🔢 독일어 숫자, 어렵지 않아요!

독일어를 시작할 때 가장 먼저 넘어야 할 산 중 하나가 바로 **숫자**죠? 0부터 10까지는 입에 착 붙을 때까지 연습하고, 그 이후의 규칙만 이해하면 수만 단위의 숫자도 거침없이 말할 수 있게 된답니다. 자, 그럼 하나씩 살펴볼까요?

---

### 1. 기본 중의 기본: 0 ~ 12

이 구간은 규칙이 따로 없어서 노래를 부르듯 외워두는 게 가장 좋아요! 특히 **1(eins)**, **2(zwei)**, **3(drei)**는 정말 자주 쓰이니 꼭 기억하세요.

| 숫자 | 독일어 (Zahl) | 읽기   |
| :--- | :------------ | :----- |
| 0    | null          | 눌     |
| 1    | eins          | 아인스 |
| 2    | zwei          | 츠바이 |
| 3    | drei          | 드라이 |
| 4    | vier          | 피어   |
| 5    | fünf          | 퓬프   |
| 6    | sechs         | 젝스   |
| 7    | sieben        | 지벤   |
| 8    | acht          | 아흐트 |
| 9    | neun          | 노인   |
| 10   | zehn          | 첸     |
| 11   | elf           | 엘프   |
| 12   | zwölf         | 츠뵐프 |

---

### 2. 10단위 만들기: 13 ~ 19

여기서부터는 규칙이 등장합니다! **[숫자 + zehn]** 조합이에요. 하지만 16과 17은 살짝 모양이 변하니 주의해 주세요!

| 숫자   | 독일어 (Zahl) | 특징                           |
| :----- | :------------ | :----------------------------- |
| 13     | dreizehn      | drei + zehn                    |
| 14     | vierzehn      | vier + zehn                    |
| 15     | fünfzehn      | fünf + zehen                   |
| **16** | **sechzehn**  | sechs에서 **-s가 탈락**해요!   |
| **17** | **siebzehn**  | sieben에서 **-en이 탈락**해요! |
| 18     | achtzehn      | acht + zehn                    |
| 19     | neunzehn      | neun + zehn                    |

---

### 3. 십 단위 숫자들: 20 ~ 90

십 단위는 보통 숫자 뒤에 **-zig**가 붙어요. 발음은 '찌히' 또는 '찌크'에 가깝답니다.

| 숫자 | 독일어 (Zahl) | 주의사항                  |
| :--- | :------------ | :------------------------ |
| 20   | zwanzig       | 츠반찌히 (불규칙)         |
| 30   | **dreißig**   | 유일하게 **-ßig**를 써요! |
| 40   | vierzig       | 피어찌히                  |
| 50   | fünfzig       | 퓬프찌히                  |
| 60   | sechzig       | -s 탈락                   |
| 70   | siebzig       | -en 탈락                  |
| 80   | achtzig       | 아흐트찌히                |
| 90   | neunzig       | 노인찌히                  |

---

### 4. ⚠️ 독일어 숫자의 꽃: "일의 자리부터 읽기"

독일어 숫자의 가장 큰 특징! 21부터는 **[일의 자리 + und(그리고) + 십의 자리]** 순서로 거꾸로 읽어요.

> 예: 21 = 1(ein) + und(그리고) + 20(zwanzig) = **einundzwanzig**

| 숫자 | 독일어 (Zahl)  | 구성        |
| :--- | :------------- | :---------- |
| 21   | einundzwanzig  | 1 그리고 20 |
| 22   | zweiundzwanzig | 2 그리고 20 |
| 35   | fünfunddreißig | 5 그리고 30 |
| 48   | achtundvierzig | 8 그리고 40 |

---

### 💡 보너스: 큰 숫자 읽기

큰 숫자는 우리가 아는 순서대로 읽으면 돼요!

- **100:** (ein)hundert [훈데르트]
- **1,000:** (ein)tausend [타우젠트]

**예시:** 125를 읽고 싶다면?
`hundert(100)` + `fünfundzwanzig(25)` = **hundertfünfundzwanzig**
