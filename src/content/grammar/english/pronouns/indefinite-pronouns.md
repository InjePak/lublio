# 부정대명사 (Indefinite Pronouns)

안녕하세요! 영어로 말하다 보면 "그것"이라고 딱 지집어 말하기보다 "아무거나 하나", "어떤 사람들"처럼 막연하게 대상을 가리켜야 할 때가 있죠? 이렇게 정해지지 않은(不定) 대상을 가리키는 말을 **부정대명사**라고 해요. 가장 많이 쓰이는 친구들을 중심으로 알아볼까요?

---

## 1. 하나를 대신하는 One

앞에 나온 명사와 종류는 같지만, '바로 그 물건'이 아닌 '같은 종류의 아무것'을 가리킬 때 사용해요.

| 구분         | 역할                        | 예시                                                            |
| ------------ | --------------------------- | --------------------------------------------------------------- |
| 단수 (one)   | 막연한 하나를 대신함        | I lost my pen. I need to buy **one**. (아무 펜이나 하나)        |
| 복수 (ones)  | 막연한 여러 개를 대신함     | I like red apples, but he likes green **ones**. (초록색 사과들) |
| 일반인 (one) | 일반적인 사람들을 나타낼 때 | **One** should obey the law. (사람은 법을 지켜야 한다.)         |

---

## 2. 수량을 나타내는 Some과 Any

두 단어 모두 '몇몇, 약간'이라는 뜻을 가졌지만, 문장의 성격에 따라 선택이 달라져요.

| 구분 | 주로 쓰이는 문장                | 예시                                                         |
| ---- | ------------------------------- | ------------------------------------------------------------ |
| some | 긍정문, 권유/부탁의 의문문      | I have **some** friends. / Would you like **some** water?    |
| any  | 부정문, 일반적인 의문문, 조건문 | I don't have **any** money. / Do you have **any** questions? |

---

## 3. 짝꿍으로 외우는 Other와 Another

두 개 혹은 세 개 이상의 대상 중에서 순서대로 가리킬 때 쓰는 표현들이에요. 그림을 그리듯 상상하며 외우면 쉬워요!

| 상황                | 표현 조합                       | 의미                                       |
| ------------------- | ------------------------------- | ------------------------------------------ |
| 대상이 둘일 때      | one ..., the other              | 하나는 ~, (나머지) 하나는 ...              |
| 대상이 셋일 때      | one ..., another ..., the other | 하나는 ~, 또 하나는 ~, (마지막) 하나는 ... |
| 일부와 나머지 전체  | some ..., the others            | 몇몇은 ~, 나머지 전부(들)는 ...            |
| 일부와 또 다른 일부 | some ..., others                | 어떤 이들은 ~, 또 다른 이들은 ...          |

---

## 4. -thing, -one, -body 시리즈

부정대명사는 `some`, `any`, `no`, `every`가 뒤에 `-thing`, `-one`, `-body`와 결합하여 더 구체적인 의미를 만들어요.

| 결합 단어 | -thing (사물)           | -one / -body (사람)           |
| --------- | ----------------------- | ----------------------------- |
| some-     | something (무언가)      | someone / somebody (누군가)   |
| any-      | anything (어느 것이든)  | anyone / anybody (누구든)     |
| no-       | nothing (아무것도 없음) | no one / nobody (아무도 없음) |
| every-    | everything (모든 것)    | everyone / everybody (모두)   |

> **⚠️ 주의:** 이 시리즈들은 항상 **단수 취급**을 해요. 그래서 동사 뒤에 `-s`를 붙이거나 `is/was`를 쓴답니다!

### 💡 학습 팁

**"One vs It, 헷갈리지 마세요!"**

- **It**: 바로 앞에서 말한 **'바로 그 물건'** (The very thing)
- **One**: 앞에서 말한 것과 같은 종류의 **'아무 물건'** (Any one of the kind)

예를 들어, "내 폰 잃어버렸어. 그거(It) 찾아야 해."와 "내 폰 잃어버렸어. 하나(One) 사야겠어."의 차이라고 생각하면 완벽해요! 오늘도 유익한 시간이 되셨길 바라요. 궁금한 점은 언제든 질문해 주세요! 😊
