# 형용사의 어순 (Order of Adjectives)

안녕하세요! 영어로 말을 하다 보면 "그의 작고 오래된 빨간색 이탈리아제 스포츠카"처럼 한 가지 물건에 여러 가지 설명을 붙이고 싶을 때가 있죠? 영어에서는 형용사를 나열할 때 약속된 순서가 있어요. 이 순서가 바뀌면 문장이 어색하게 들릴 수 있답니다. 루블리오와 함께 그 마법의 순서를 알아봐요!

---

## 1. 형용사 나열의 황금률

명사 앞에 오는 형용사들은 보통 다음과 같은 순서로 배치됩니다. 명사에서 멀어질수록 주관적인 의견이, 명사와 가까울수록 객관적이고 변하지 않는 성질이 위치해요.

| 순서 | 카테고리 (Category) | 설명                          | 예시                            |
| ---- | ------------------- | ----------------------------- | ------------------------------- |
| 1    | Opinion (의견)      | 예쁘다, 맛있다 등 주관적 생각 | beautiful, delicious, nice      |
| 2    | Size (크기)         | 크다, 작다, 길다 등           | big, small, tall, long          |
| 3    | Age (나이/신구)     | 새것, 헌것, 젊은, 오래된      | new, old, young, ancient        |
| 4    | Shape (모양)        | 둥근, 네모난, 납작한          | round, square, flat             |
| 5    | Color (색깔)        | 빨간, 푸른, 검은              | red, blue, dark, yellowish      |
| 6    | Origin (출신/국적)  | 한국의, 미국인의, 수입산      | Korean, American, French        |
| 7    | Material (재료)     | 나무로 된, 금속의, 면의       | wooden, metal, cotton, plastic  |
| 8    | Purpose (목적)      | ~용 (명사 형태가 많음)        | sleeping (bag), walking (stick) |

---

## 2. 한눈에 보는 어순 예시

실제 문장에서 어떻게 쓰이는지 볼까요? 모든 단계를 다 쓰기보다는 보통 2~3개의 형용사를 조합해서 사용해요.

| 조합               | 예시 문장                         | 어순 분석                |
| ------------------ | --------------------------------- | ------------------------ |
| 크기 + 색깔        | a **large blue** box              | Size + Color             |
| 의견 + 모양        | a **lovely round** table          | Opinion + Shape          |
| 신구 + 국적 + 재료 | an **old Italian leather** jacket | Age + Origin + Material  |
| 의견 + 크기 + 목적 | a **wonderful big swimming** pool | Opinion + Size + Purpose |

---

## 3. 예외와 주의사항

기본 순서 외에도 꼭 알아두어야 할 포인트가 몇 가지 더 있어요!

| 구분                 | 내용                                                           | 예시                         |
| -------------------- | -------------------------------------------------------------- | ---------------------------- |
| 한정사 (Determiners) | 관사(a, the), 소유격(my), 지시어(this)는 항상 맨 앞에 와요.    | **My** expensive old car.    |
| 수량 형용사          | 개수(one, two)나 서수(first, second)는 성질 형용사보다 앞서요. | The **two** beautiful girls. |
| 같은 카테고리        | 같은 성격의 형용사를 나열할 땐 `and`를 쓰거나 쉼표를 찍어요.   | A **black and white** cat.   |

### 💡 학습 팁

이 긴 순서를 다 외우기 힘들다면, 앞 글자만 따서 입으로 익혀보세요!
**"의·크·노·모·색·출·재·목"** (의견, 크기, 노소, 모양, 색깔, 출신, 재료, 목적)

이렇게 외워두면 나중에 긴 문장을 만들 때도 당황하지 않고 형용사를 배치할 수 있을 거예요. 특히 **의견**이 가장 먼저 나오고, **재료**나 **목적**이 명사 바로 앞에 붙는다는 것만 기억해도 절반은 성공입니다!
