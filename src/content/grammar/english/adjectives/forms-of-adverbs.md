# 부사의 형태 (Forms of Adverbs)

안녕하세요! 영어에서 부사는 "어떻게(방법)", "언제(시간)", "어디서(장소)" 등을 나타내며 동사, 형용사, 혹은 다른 부사를 꾸며주는 역할을 해요. 대부분의 부사는 형용사의 꼬리를 살짝 바꿔서 만들지만, 독특한 모양을 가진 친구들도 있답니다. 루블리오와 함께 부사의 변신 규칙을 알아봐요!

---

## 1. 형용사 + -ly 규칙

가장 일반적인 부사 제조법이에요. 형용사 끝 모양에 따라 규칙이 조금씩 달라집니다.

| 형용사 끝 모양 | 변화 규칙            | 예시 (형용사 → 부사)                         |
| -------------- | -------------------- | -------------------------------------------- |
| 일반적인 경우  | + -ly                | quick → quickly, slow → slowly               |
| 자음 + y       | y를 i로 고치고 + -ly | happy → happily, easy → easily               |
| -le로 끝남     | e를 빼고 + -y        | gentle → gently, simple → simply             |
| -ue로 끝남     | e를 빼고 + -ly       | true → truly                                 |
| -ic로 끝남     | + -ally              | fantastic → fantastically, magic → magically |

---

## 2. 형용사와 부사의 모양이 같은 경우

단어 하나가 형용사와 부사 역할을 모두 하는 친구들이에요. 문장에서 어떤 단어를 꾸미고 있는지에 따라 구분해야 해요.

| 단어  | 형용사 의미    | 부사 의미    | 예시 문장                                        |
| ----- | -------------- | ------------ | ------------------------------------------------ |
| fast  | 빠른           | 빠르게       | He is a **fast** runner. / He runs **fast**.     |
| hard  | 어려운, 딱딱한 | 열심히, 몹시 | It is a **hard** work. / I study **hard**.       |
| late  | 늦은           | 늦게         | I was **late** for school. / I arrived **late**. |
| high  | 높은           | 높게         | A **high** mountain. / The bird flies **high**.  |
| early | 이른           | 일찍         | An **early** bird. / I woke up **early**.        |

---

## 3. -ly가 붙으면 뜻이 완전히 달라지는 부사

모양이 같았던 친구들에게 `-ly`를 붙이면, 원래 의미와는 전혀 다른 새로운 뜻이 생기기도 해요. 이 부분은 시험에도 자주 나오니 꼭 체크하세요!

| 단어  | 의미   | -ly 형태 부사 | 의미              |
| ----- | ------ | ------------- | ----------------- |
| hard  | 열심히 | hardly        | 거의 ~않다 (부정) |
| late  | 늦게   | lately        | 최근에            |
| high  | 높게   | highly        | 매우, 대단히      |
| near  | 가까이 | nearly        | 거의              |
| short | 짧게   | shortly       | 곧, 즉시          |

---

## 4. 주의해야 할 형용사 모양

단어 끝이 `-ly`로 끝나서 부사처럼 보이지만, 사실은 **형용사**인 단어들이 있어요. 이들은 부사로 쓰려면 특별한 표현이 필요해요.

| 단어     | 의미 (형용사) | 부사로 쓰려면?               |
| -------- | ------------- | ---------------------------- |
| friendly | 친절한        | in a friendly way (친절하게) |
| lovely   | 사랑스러운    | in a lovely way (사랑스럽게) |
| lonely   | 외로운        | in a lonely way (외롭게)     |
| silly    | 어리석은      | in a silly way (어리석게)    |

---

### 💡 학습 팁

**"Good vs Well"**
가장 많이 실수하는 부사가 바로 `good`의 부사형이에요. `goodly`라는 말은 거의 쓰이지 않아요!

- **Good**: 형용사 (좋은) → "You are a **good** singer."
- **Well**: 부사 (잘) → "You sing **well**."

단, `well`이 '건강한'이라는 뜻의 형용사로 쓰일 때도 있다는 점까지 기억하면 당신은 이미 부사 마스터! 오늘 배운 규칙들을 활용해서 더 풍성한 문장을 만들어 보세요. 궁금한 점은 언제든 질문해 주세요! 😊
