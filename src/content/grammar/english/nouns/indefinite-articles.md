# 부정관사 (Indefinite Articles: A, An)

안녕하세요! 영어에서 명사를 쓰기 전에 그 정체가 무엇인지 살짝 힌트를 주는 친구들이 있죠? 바로 **관사**입니다. 그중에서도 "정해지지 않은 아무개 하나"를 가리킬 때 쓰는 **부정관사 a와 an**에 대해 친절하게 알려드릴게요!

---

## 1. a와 an의 기본 구분법

`a`를 쓸지 `an`을 쓸지는 바로 뒤에 오는 단어의 **첫 소리(발음)**에 달려 있어요. 철자가 아니라는 점이 정말 중요해요!

| 구분 | 사용 조건                            | 예시                      |
| ---- | ------------------------------------ | ------------------------- |
| a    | 자음 소리(b, c, d, f...)로 시작할 때 | a book, a cat, a user     |
| an   | 모음 소리(a, e, i, o, u)로 시작할 때 | an apple, an egg, an hour |

---

## 2. 주의해야 할 발음 예외

눈으로 보는 철자와 귀로 듣는 소리가 다를 때가 있어요. 시험이나 실전 회화에서 자주 실수하는 부분이니 꼭 체크해 두세요!

| 예외 상황                         | 이유                                       | 올바른 예시                         |
| --------------------------------- | ------------------------------------------ | ----------------------------------- |
| 철자는 u인데 a를 쓰는 경우        | 'u'가 [ju:] 자음 소리로 시작함             | a university, a unit, a uniform     |
| 철자는 h인데 an을 쓰는 경우       | 'h'가 묵음(소리 안 남)이라 모음으로 시작함 | an hour, an honest man              |
| 철자는 m, s, f인데 an을 쓰는 경우 | 알파벳 첫 소리가 모음(에-)으로 시작함      | an MP3 player, an SMS, an FBI agent |

---

## 3. 부정관사는 언제 사용할까요?

부정관사는 단순히 '하나'라는 뜻 외에도 여러 가지 상황에서 쓰인답니다.

| 쓰임새             | 설명                                       | 예시                                                  |
| ------------------ | ------------------------------------------ | ----------------------------------------------------- |
| 정해지지 않은 하나 | 특별한 대상이 아닌 아무거나 하나를 말할 때 | I need a pencil. (연필 아무거나 하나 필요해.)         |
| 종족 전체 대표     | 그 종류에 속하는 일반적인 전체를 말할 때   | A dog is a faithful animal. (개라는 동물은 충직하다.) |
| 직업/신분          | 사람의 직업이나 정체를 소개할 때           | She is a doctor. (그녀는 의사이다.)                   |
| '~당' (per)        | 횟수나 속도 등 단위를 나타낼 때            | twice a week (일주일에 두 번)                         |

---

## 4. 부정관사를 쓸 수 없는 경우

부정관사는 이름 그대로 '셀 수 있는 명사'가 '하나'일 때만 친해질 수 있어요.

| 불가 상황   | 이유                                      | 예시                    |
| ----------- | ----------------------------------------- | ----------------------- |
| 복수 명사   | 여러 개일 때는 a/an을 쓸 수 없어요.       | a books (X) → books (O) |
| 불가산 명사 | 셀 수 없는 물질이나 개념에는 못 써요.     | a water (X) → water (O) |
| 고유 명사   | 세상에 하나뿐인 이름 앞에는 보통 안 써요. | a Seoul (X), a Tom (X)  |

---

### 💡 학습 팁

**"A university인가요, An university인가요?"**
많은 분이 `u`가 모음이라서 `an`이라고 생각하시지만, 발음 기호를 보면 [j]라는 자음 소리로 시작해요. 그래서 **`a university`**가 정답입니다! 관사를 결정할 때는 항상 입으로 직접 소리 내어 읽어보는 습관을 지녀보세요.
