# 명사의 수 (Number of Nouns)

안녕하세요! 영어에서 명사가 하나인지(단수), 여러 개인지(복수)를 구분하는 것은 아주 중요해요. 우리말과 달리 영어는 개수에 따라 단어의 형태가 바뀌기 때문이죠. 규칙만 알면 생각보다 간단하답니다. 함께 알아볼까요?

---

## 1. 규칙적으로 변하는 복수형

가장 기본적으로는 명사 뒤에 `-s`나 `-es`를 붙여요. 끝나는 철자에 따라 규칙이 조금씩 다르니 아래 표를 확인해 보세요!

| 규칙                       | 만드는 방법                  | 예시                                      |
| -------------------------- | ---------------------------- | ----------------------------------------- |
| 일반적인 경우              | 명사 끝에 -s 붙이기          | dog → dogs, book → books                  |
| -s, -x, -ch, -sh로 끝날 때 | 명사 끝에 -es 붙이기         | bus → buses, box → boxes, watch → watches |
| 자음 + y로 끝날 때         | y를 i로 바꾸고 -es 붙이기    | baby → babies, city → cities              |
| f, fe로 끝날 때            | f/fe를 v로 바꾸고 -es 붙이기 | leaf → leaves, knife → knives             |
| 자음 + o로 끝날 때         | 명사 끝에 -es 붙이기         | tomato → tomatoes, hero → heroes          |

---

## 2. 불규칙하게 변하는 복수형

가끔 규칙을 무시하고 자기 마음대로 변하는 명사들도 있어요. 이 친구들은 자주 보면서 눈에 익히는 것이 가장 좋아요!

| 유형                      | 단수형 (Singular) | 복수형 (Plural) |
| ------------------------- | ----------------- | --------------- |
| 모음이 변하는 경우        | man               | men             |
| 모음이 변하는 경우        | foot              | feet            |
| 모음이 변하는 경우        | tooth             | teeth           |
| 형태가 완전히 변하는 경우 | child             | children        |
| 형태가 완전히 변하는 경우 | mouse             | mice            |

---

## 3. 단수와 복수가 같은 명사

세상에는 하나일 때와 여러 개일 때의 모습이 똑같은 명사들도 있답니다. 헷갈리기 쉽지만, 문맥을 보고 판단할 수 있어요.

| 명사 종류      | 예시    | 참고 사항                                     |
| -------------- | ------- | --------------------------------------------- |
| 동물의 이름    | sheep   | 양 한 마리도 sheep, 열 마리도 sheep           |
| 동물의 이름    | deer    | 사슴 한 마리도 deer, 여러 마리도 deer         |
| 동물의 이름    | fish    | 종류를 강조할 때 외에는 보통 fish 그대로 사용 |
| 학문/국가 관련 | Chinese | 중국인 한 명도, 여러 명도 Chinese             |

---

## 4. 항상 복수형으로 쓰는 명사

물건 하나라도 조각이 두 개가 모여 하나의 기능을 하는 것들은 항상 `-s`를 붙여서 복수형으로 써요.

| 카테고리  | 예시     | 특징                        |
| --------- | -------- | --------------------------- |
| 도구/장비 | scissors | 가위 (날이 두 개)           |
| 의류/안경 | glasses  | 안경 (알이 두 개)           |
| 의류/안경 | pants    | 바지 (다리 부분이 두 개)    |
| 의류/안경 | socks    | 양말 (보통 두 짝이 한 세트) |

---

### 💡 학습 팁

- **y로 끝나도 다 같은 게 아니에요!** `boy`처럼 **모음(a, e, i, o, u) + y**로 끝날 때는 그냥 `-s`만 붙여서 `boys`가 된답니다. `i`로 바꾸지 않도록 주의하세요!
- **항상 복수인 명사**를 셀 때는 `a pair of` (한 쌍의)라는 표현을 쓰면 아주 세련된 영어가 돼요. (예: a pair of scissors)
